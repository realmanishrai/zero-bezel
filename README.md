\# Zero-Bezel 📱📱



\*\*Zero-Bezel\*\* is an Android application that transforms multiple smartphones into a single, unified extended display. By bridging devices over a local Wi-Fi network or mobile hotspot, Zero-Bezel eliminates the need for expensive monitors or tablets, allowing you to combine 2, 3, or even 5 phones into one giant, continuous screen.



Whether you need a wider canvas for web browsing, a synchronized media player for videos, or a massive dashboard for PDFs, Zero-Bezel handles the heavy lifting of viewport splitting, touch-event forwarding, and ultra-low-latency rendering.



\## 🚀 The Vision

We believe everyone has a drawer full of old or secondary phones. Zero-Bezel turns that e-waste into a powerful, multi-screen productivity and entertainment hub. 



\## 🛠️ Current Scope (Phase 1)

\- \*\*Android-to-Android:\*\* Clustering multiple Android phones into a single logical display.

\- \*\*Host/Client Architecture:\*\* One device acts as the "Brain" (Host) while others act as "Monitors" (Clients).

\- \*\*Local Networking:\*\* Zero internet required for the core display extension (uses local Wi-Fi Direct / Hotspot).

\- \*\*Built-in Ecosystem:\*\* Custom wide-canvas browser, synchronized media player, and PDF viewer.
